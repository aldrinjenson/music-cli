# music-cli(WIP)

Search for any song of your choice and have them downloaded locally using `youtbe-dl`

![demo.png](demo.png)

## Requirements

- nodeJs & npm
- [youtube-dl](https://github.com/ytdl-org/youtube-dl)

## Installation

- clone this repo

```bash
npm install
```

## Usage

```bash
./musica.sh <search_query>
```
